# Онбординг клиента — Whale Digital Landing Pages

## Как это работает

Ваш лендинг — это один JSON-файл. Вы заполняете его как анкету, мы публикуем сайт за 24 часа.

---

## Шаг 1 — Домен

Скажите нам, какой домен вы хотите использовать (например, `yourbrand.com`).

Если домена ещё нет — помогаем зарегистрировать или настроить.

---

## Шаг 2 — Заполните конфиг-файл

Мы пришлём вам файл вида `ваш-бренд.json`. Откройте его в любом текстовом редакторе (можно даже в VSCode или Notepad++) и заполните по инструкции ниже.

### Основные поля

```json
{
  "meta": {
    "title": "Ваш заголовок для Google | Ваш бренд",
    "description": "Описание для SEO и соцсетей (150-160 символов)",
    "lang": "ru"
  },
  "theme": {
    "primaryColor": "#111827"
  }
}
```

> **Цвет:** укажите HEX-код основного цвета вашего бренда (например `#E53E3E` для красного, `#2B6CB0` для синего).

### Блоки страницы

Каждый блок — это раздел лендинга. Заполните только те, что актуальны для вас:

| Блок | Что это | Обязательно? |
|------|---------|--------------|
| `hero` | Главный экран с заголовком и формой | Да |
| `pains` | «Узнаёт ли клиент себя?» — боли и проблемы | Рекомендуется |
| `socialProof` | Цифры доверия: лет в деле, клиентов, результатов | Рекомендуется |
| `solution` | Ваш подход и история | Рекомендуется |
| `features` | Что конкретно вы делаете (6 пунктов) | Рекомендуется |
| `audience` | Для кого — и для кого точно не подходит | Рекомендуется |
| `pricing` | Форматы/пакеты работы | По желанию |
| `howItWorks` | Пошаговый процесс сотрудничества | Рекомендуется |
| `testimonials` | Отзывы клиентов | По желанию |
| `team` | Команда / о вас | По желанию |
| `faq` | Частые вопросы | Рекомендуется |
| `finalCta` | Финальный призыв к действию | Да |

---

## Шаг 3 — Фото (если нужно)

Если хотите разместить фото (ваше, команды, логотип):

- Назовите файл: `photo.jpg` или `logo.png`
- Пришлите нам вместе с JSON-файлом

Поддерживаемые форматы: JPG, PNG, WebP. Рекомендуемый размер: не более 1 МБ.

**Где фото появятся на сайте:**
- `team[0].avatar` — ваша фотография в блоке «Команда»
- `solution.imageSrc` — фото в блоке «О вас / История»
- `hero.imageSrc` — фото в главном блоке (опционально)

---

## Шаг 4 — Пришлите нам

Отправьте на почту / в Telegram:
1. Заполненный `.json` файл
2. Фото (если есть)
3. Домен, на котором должен работать сайт

Мы опубликуем сайт в течение **24 часов**.

## Часто задаваемые вопросы

**Нужны ли технические знания?**
Нет. JSON — это просто текст. Если вы умеете редактировать Word — справитесь с JSON. Пример файла мы пришлём с подсказками на русском языке.

**Можно ли изменить сайт потом?**
Да, в любое время. Просто пришлите обновлённый файл — изменения вступят в силу в течение нескольких часов.

**Сколько это стоит?**
Уточняйте актуальные тарифы у менеджера. В стоимость обычно входит: публикация, хостинг, домен (если нужен), настройка аналитики и CRM-интеграция.

**Что происходит с заявками с сайта?**
Заявки автоматически попадают в вашу CRM-систему. Мы настраиваем это при публикации — вам не нужно делать ничего дополнительно.
